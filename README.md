# Simple cmd waiter
# Простой таймер cmd

To start, enter: "wait.exe time in seconds". Example: "wait.exe 5"                                   
Для запуска введите: "wait.exe время в секундах". Пример: "wait.exe 5"

If you want to access it from any directory, then place the install.bat file next to wait.exe and run it as Administrator (install.bat will copy wait.exe to system32).                                                        
Если хотите обращаться к нему из любых катологов, то положите рядом с wait.exe файл install.bat и запустите его от имени Администратора(install.bat просто скопирует wait.exe в system32).

If you want to remove it, download remove.bat and run it as Administrator (remove.bat will remove wait.exe from system32).                                             
Если хотите удалить его, скачайте remove.bat и запустите его от имени Администратора(remove.bat удалит wait.exe из system32).
